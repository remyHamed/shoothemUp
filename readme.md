## Activez l'environnement virtuel :

- **Sous Unix ou macOS** : 
  ```
  source env/bin/activate
  ```

- **Sous Windows** : 
  ```
  .\env\Scripts\Activate.ps1
  ```

## Installez les dépendances :

```
pip install -r requirements.txt
```

## mettre à jour le fichier de dépendance :

```
pip freeze > requirements.txt
```

## Exécution du jeu :

```
python index.py
```
